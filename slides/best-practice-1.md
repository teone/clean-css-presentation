#  Best Practice

## SRP

**Single Responsibility Principle** => _every module or chunk of code (a function etc) should do one job well and one job only_
<hr>
This is good:<br>
```
    .blockquote {
        background: #fff;
    }```
<br><br>
This is Bad:<br>
```
    .text-center {
        text-align: center;
        color: #ff0;
    }```
